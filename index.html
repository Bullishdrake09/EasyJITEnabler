Steps to Install

Step 1: Make sure when installing Python, it is added to path and ensure that "Install launcher for all users" is DISABLED. This is very important for it to work.

Step 2: Make sure you have the latest version of Python installed. After instalation restart your computer.

Step 3: Download EasyJITEnabler from my github

Step 4: Extract the zip file you just downloaded from my github

Step 5: Open up "pib installer here" inside the folder and wait until it is installed

Step 6: Make sure iTunes (web version!) is installed!

Step 7: Open up ENABLE SERVER FOR FIRST TIME HERE, wait until it asks you to type y/N, type y and click enter

Step 8: Install this shortcut on your iPhone (Shortcut download above is the same download link)

Step 9: When installing the shortcut, it will ask you to put in your UDID. To get this, open up iTunes on your PC/laptop, click on the phone emoji/icon at the top and click on 'serial number' once to see your UDID. Type this code into the first prompt of the SideJIT shortcut.

Step 10: The next thing it asks you is your SideJIT server address. To get this, go to the terminal (the black box with all the code) and type the last number generated (e.g., 192.168.x.x). Write that number down in the shortcut, and it is all set up. Did you close the terminal? No problem. Run "OPEN THIS FILE TO ENABLE JIT SERVER," and it will generate a new code for you. (You need to be on the same Wi-Fi network as your phone AND you have to stay connected with a cable to your computer!)

Step 11: After all of this, we can finally enable JIT. Make sure your phone is still connected to your laptop, run the "SideJIT" shortcut, select the app you want to enable JIT on, and you should get a message saying 'JIT enabled!'. Disclaimer: You do not have to do all those steps over and over again to run JIT. Running "OPEN THIS FILE TO ENABLE JIT SERVER" Is enough for the server to start, and normally, if you do not change networks, the number you wrote down will not change. (192.168....).
